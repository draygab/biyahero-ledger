# Biyahero Ledger

A blockchain-based fuel expense verification and reimbursement system for delivery riders.

## Problem
Delivery riders struggle with delayed or disputed fuel reimbursements due to lack of tamper-proof proof.

## Solution
Fuel logs are hashed and stored on Stellar via Soroban, enabling instant verification and automated XLM payouts.

## Timeline
- Day 1–2: Smart contract
- Day 3–4: Mobile UI
- Day 5: Integration + demo

## Stellar Features
- Soroban smart contracts
- XLM transfers

## Vision
To protect gig workers’ income through transparent and automated financial systems.

## Prerequisites
- Rust
- Soroban CLI
